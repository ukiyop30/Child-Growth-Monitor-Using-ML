# Child Growth Monitor Using Machine Learning

## Project Description
This project focuses on monitoring child growth trends using Machine Learning techniques and UNICEF SDG datasets. The system analyzes historical child malnutrition data and predicts future growth patterns.

## Dataset Source
UNICEF Child Malnutrition Dataset  
https://data.unicef.org

## Algorithms Used
Linear Regression

## Tools Used
Python  
Google Colab  
Pandas  
NumPy  
Matplotlib  
Seaborn  
Scikit-learn

## Objectives
- Analyze child stunting trends
- Predict future growth patterns
- Support SDG 2 and SDG 3 initiatives

## Output
The system generates:
- Growth trend graphs
- Prediction results
- Model accuracy scored
